# Back-end-Services
To ensure that my code, program, and software is funstional and secure. I reviewed the requirements and created the tests to make sure all was behaving properly. While developing the application I used Junit testing and coverage analysis to get as much test coverage as possible. This helped me able to find and clear out bugs early in the development process. To interpret the users needs I read the requirements and break each one done into a task I would make then test then move onto the next one. I approach designing software by getting an understanding of what the requirements are and what problems the user wants solved. Once I undersatnd the requirements I can then design the application based on the users needs.
